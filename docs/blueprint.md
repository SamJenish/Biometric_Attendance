# **App Name**: Biometric Presence Pro

## Core Features:

- Biometric Authentication: Authenticate users (students and admins) using device biometrics (fingerprint or face unlock).
- GPS Location Validation: Validate the user's GPS location against predefined classroom coordinates.
- Attendance Recording: Record student attendance data, including student ID, subject, timestamp, latitude, and longitude in Firestore.
- Admin Authentication: Admin user authenticates via email and password.
- Attendance Reporting Tool: Generate reports of the class attendace on a given day, and display results, which can be used by an admin to identify trends.
- Real-time Attendance Dashboard: Display attendance records in real-time for admin users with latest records listed at the top.
- Secure Timestamping: Timestamp attendance records using Firestore server time to prevent manipulation.

## Style Guidelines:

- Primary color: Deep Navy (#2E3A59) for a professional and modern feel.
- Accent color: Mint Green (#00C2A8) for interactive elements and highlights.
- Background color: Soft Grey (#F4F6FA) to provide a clean and minimal base.
- Font: 'Poppins' sans-serif for both headings and body text, lending an academic, futuristic elegance.
- Utilize a card-based UI to present attendance records in an organized and easily digestible manner.
- Use simple and consistent icons for navigation and actions, avoiding clutter and enhancing clarity.
- Incorporate subtle animations, such as a soft glow around the biometric button, to enhance the user experience without being distracting.
